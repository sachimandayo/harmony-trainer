# コードフラッシュカード

12キーのダイアトニックコードとⅡ–Ⅴ–Ⅰを、日本語のフラッシュカード形式で練習するWebアプリです。

## 使い方

`index.html` をブラウザで開くだけで使えます。サーバー、ログイン、データベースは不要です。

1. モード、対象キー、練習枚数を選びます。
2. 表面の問題を見て、頭の中で答えを考えます。
3. カードをクリックまたはタップして裏返します。
4. 答えを見て「覚えていた」または「まだ不安」を押します。

パソコンでは次のキー操作も使えます。

- Space / Enter: カードを裏返す
- 右矢印: 覚えていた
- 左矢印: まだ不安

## 学習モード

- メジャーダイアトニック
- ナチュラルマイナー
- ハーモニックマイナー
- メロディックマイナー
- メジャーⅡ–Ⅴ–Ⅰ
- マイナーⅡ–Ⅴ–Ⅰ
- メジャーカデンツ
- マイナーカデンツ
- すべて混合

メロディックマイナーは、ジャズで一般的な上行形を使用しています。
カデンツは、メジャーキーでは `Ⅵ–Ⅱ–Ⅴ–Ⅰ`、マイナーキーでは `Ⅳm–Ⅱm7♭5–Ⅴ7–Ⅰm` を使用しています。

## 学習記録

学習記録はブラウザの `localStorage` に保存されます。カードごとに表示回数、自己評価回数、自己評価率、最終学習日時、連続「覚えていた」回数を記録します。

学習記録画面では、総学習枚数、モード別、キー別、度数別、苦手カード上位10枚を確認できます。

## ファイル構成

- `index.html`: 画面構造
- `style.css`: デザインとレスポンシブ対応
- `script.js`: カード生成、学習進行、記録保存、検証処理

## 検証

`script.js` の起動時に、指定された代表例のコード一覧とⅡ–Ⅴ–Ⅰを `console.assert` で検証します。コンソールにエラーが出なければ、検証ケースは通過しています。
